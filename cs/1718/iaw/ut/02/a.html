IAW - UT2 - A

ENTREGA

• Desde la carpeta de usuario en Dropbox, crea una carpeta UT (las dos letras en mayúsculas) y dentro de ella crea una carpeta 02.
• Dentro de la carpeta 02 sube un archivo zip (máximo 10 MB) con la práctica elegida.
• La práctica debe estar subida en la plataforma Heroku y accesible desde internet.
• Todos los documentos tienen que estar validados en XHTML 5.2, WCAG 2.0 AAA y CSS3.

PRÁCTICA

Crea una web dinámica (XHTML + CSS + PHP o Python) que genere tanto un documento HTML final como un PDF final.

Elige uno de los siguientes temas:

GENERADOR DE CV

El usuario introducirá sus datos personales, su experiencia profesional, su formación, dominio de idiomas y otros datos que sean de interés. La aplicación web generará un CV tanto en un archivo HTML (incluir el estilo dentro del archivo HTML) como un PDF.

INFORME DE SALUD PERSONAL

El usuario introducirá su edad, altura, peso, medidas, actividad y todos los datos necesarios. La aplicación web generará un informe de salud tanto en HTML (incluir el estilo dentro del archivo HTML) como en PDF. Un ejemplo puede encontrarse aquí.

INFORME DE EVALUACIÓN DE UN ESTUDIANTE

El usuario introducirá sus datos personales y todos los puntos obtenidos en cada CE (criterio de evaluación). La aplicación web calculará la calificación final y mostrará todos los detalles de dicha calificación. Dicho informe se generará tanto en HTML (incluir el estilo dentro del archivo HTML) como en PDF. Las tablas para calcular la calificación final puede encontrarse aquí.
