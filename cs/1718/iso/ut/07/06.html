Supervisión del rendimiento del sistema

Contenidos

  1. Herramientas de monitorización en Windows 10:
    • Task Manager (taskmgr).
    • Event Viewer (eventvwr).
    • Performance Monitor (perfmon).
    • Reliability Monitor (perfmon /rel).
    • Resource Monitor (resmon).
  2. Herramientas de monitorización en GNU/Linux:
    • uptime.
    • time.
    • top.
    • ps.
    • vmstat.
    • free.
    • who.
    • w.

Actividades

Realiza las siguientes actividades en Windows 10 y/o en GNU/Linux y documéntalo en un PDF:

01. Identifica los diferentes tipos de procesos, trabajos (jobs) y/o eventos del sistema.

02. Utiliza herramientas de monitorización en tiempo real.

03. Monitoriza el rendimiento del sistema mediante registros de contador usando Performance Monitor.

04. Planifica y configura alertas de rendimiento usando Data Collector Sets en Performance Monitor.

05. Interpreta los registros/alertas de rendimiento creados en la actividad anterior.

06. Analiza el sistema para optimizar el rendimiento.

07. Genera documentación de las posibles incidencias del sistema..
