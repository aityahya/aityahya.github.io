Actividades

Realiza las siguientes actividades en Windows 10 y/o en GNU/Linux y documéntalo en un PDF:

1. Instala un software específico según la documentación que proporciona su página oficial.

2. Describe cómo realizarías instalaciones desatendidas en red de una distribución GNU/Linux como Kali Linux con PXE y de Windows 10 con WDS.

3. Explica cómo configurar y utilizar un servidor de actualizaciones automáticas en Windows Server 2016 para diferentes clientes con Windows 10.

4. Planifica un protocolo de actuación ante incidencias software.

5. Detalla los pasos a seguir en el protocolo de actuación anterior para resolver incidencias software.

6. Efectúa una asistencia técnica a través de la red resolviendo alguna incidencia software.

7. Elabora un breve manual o una breve guía visual para enseñar a utilizar un sistema operativo o aplicación.
