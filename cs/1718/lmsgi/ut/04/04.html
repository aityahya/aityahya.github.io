RA04

Establece mecanismos de validación para documentos XML utilizando métodos para definir su sintaxis y estructura.

  • XML: desde HOME hasta XML Namespaces.
  • Teoría: XML Schema Definition (XSD).
  • Referencia: XML Schema Definition (XSD).
  • Ejemplos: XML, XSD.

Entrega:

  • Todos los documentos XML y XSD tienen que estar bien formados (Well-Formed) y el XSD además tiene que ser válido (Validity). Una vez terminado, el documento XML tiene que validarse correctamente con respecto al XSD creado. Para realizar dichas operaciones, puedes usar esta herramienta en línea, el plugin XML Tools de Notepad++ o la aplicación XML Copy Editor.
  • Desde la carpeta de usuario en Dropbox, crea una carpeta RA y dentro de ella crea una carpeta 04.
  • Dentro de la carpeta 04 sube los ejercicios con el siguiente formato: 01.xsd, 01a.xml, 01b.xml, 01c.xml, 02.xsd, 02a.xml, 02b.xml, 02c.xml, etc.
  • La fecha límite de entrega es el viernes 21 de abril de 2017 a las 23.59 horas.

Actividades

Crea un XSD (XML Schema Definition) válido que defina las siguientes estructuras. Además, realiza 3 documentos XML de ejemplo y valídalos con respecto al XSD válido creado:

01. Define la estructura de un correo electrónico.

02. Crea un XML Schema que describa la estructura de un estudiante. La estructura debe contener elementos simples que contemplen, al menos, 5 tipos diferentes de datos y valores fijos y por defecto. Realiza 3 documentos XML de ejemplo y valídalos con respecto al XSD creado. Pista: usar los ejemplos de W3Schools y la teoría de XSD Data.

03. Crea un XML Schema que describa la estructura taxonómica de un perro. La estructura debe contener elementos simples con atributos, que contemplen ambos, al menos, 5 tipos diferentes de datos y valores fijos, por defecto y obligatorios. Realiza 3 documentos XML de ejemplo y valídalos con respecto al XSD creado. Pista: usar los ejemplos de W3Schools.

04. Crea un XML Schema que describa la estructura de un CD. La estructura debe contener elementos simples con atributos, que contemplen ambos, al menos, 5 tipos diferentes de datos y valores fijos, por defecto y obligatorios. Además, la estructura debe contener 5 restricciones. Por último, debe contener, como mínimo, una anotación. Realiza 3 documentos XML de ejemplo y valídalos con respecto al XSD creado. Pista: usar los ejemplos de W3Schools.

05. Crea un XML Schema que describa la estructura de un producto. La estructura debe contener los 4 tipos de elementos complejos. Además, la estructura debe contener 5 restricciones. Realiza 3 documentos XML de ejemplo y valídalos con respecto al XSD creado. Pista: usar los ejemplos de elementos complejos de W3Schools: empty, only elements, only text y mixed.

06. Crea un XML Schema que describa la estructura de un catálogo de CDs. Para la estructura del CD puedes basarte en la realizada en la actividad 04. El catálogo debe contener 7 indicadores. Realiza 3 documentos XML de ejemplo y valídalos con respecto al XSD creado. Pista: usar los ejemplos de W3Schools.

07. Crea un XML Schema que describa la estructura de una persona (nombre, apellidos, etc.). El esquema debe contener los elementos <any> y <anyAttribute>. Además, algunos nombres de los elementos podrán estar tanto en inglés como en español. Realiza 3 documentos XML de ejemplo y valídalos con respecto al XSD creado. Pista: usar los ejemplos de W3Schools de any, anyAttribute y los de elementos de sustitución para los idiomas.

08. Crea un XML Schema que describa la estructura de un catálogo de libros. La estructura debe contener elementos complejos con atributos, que contemplen ambos, al menos, 5 tipos diferentes de datos y valores fijos, por defecto y obligatorios. Además, la estructura debe contener 5 restricciones y 7 indicadores. El esquema debe contener los elementos <any> y <anyAttribute>. Además, algunos nombres de los elementos podrán estar tanto en inglés como en español. Por último, debe contener, como mínimo, una anotación. Realiza 3 documentos XML de ejemplo y valídalos con respecto al XSD creado. Pista: usar todos los ejemplos de XML Schema Definition (XSD) de W3Schools.
