• La práctica debe estar subida en la plataforma Heroku y accesible desde internet.
• Todos los documentos tienen que estar validados en XHTML 5.2, WCAG 2.0 AAA y CSS3.

PRÁCTICA

Crea una web dinámica (XHTML + CSS + PHP o Python) que genere tanto un documento HTML final como un PDF final.

Elige uno de los siguientes temas:

Generador de CV

El usuario introducirá sus datos personales, su experiencia profesional, su formación, dominio de idiomas y otros datos que sean de interés. La aplicación web generará un CV tanto en HTML (más CSS) como un PDF.

Informe de salud personal

El usuario introducirá su edad, altura, peso, medidas, actividad y todos los datos necesarios. La aplicación web generará un informe de salud tanto en HTML (más CSS) como en PDF. Un ejemplo puede encontrarse aquí.

Informe de evaluación de un estudiante

El usuario introducirá sus datos personales y todos los puntos obtenidos en cada CE (criterio de evaluación). La aplicación web calculará la calificación final y mostrará todos los detalles de dicha calificación. Dicho informe se generará tanto en HTML (más CSS) como en PDF. Las tablas para calcular la calificación final puede encontrarse aquí.
