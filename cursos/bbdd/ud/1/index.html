Contenidos

  • Elementos de las bases de datos.

Actividades

Base de datos

1. ¿Qué es una Base de Datos (BBDD)?

2. ¿Cuántos modelos de bases de datos existen? ¿Cuál modelo es el más usado actualmente? Por último, detalla cinco ejemplos del mundo real donde se usen bases de datos relacionales.

3. ¿En qué se diferencia una base de datos de una hoja de cálculo? Investígalo e indica las ventajas de usar bases de datos sobre hojas de cálculo.

Sistema Gestores de Bases de Datos

4. ¿Qué es un Sistema Gestor de Bases de Datos (SGBD)?

5. Indica las funciones, componentes, arquitectura y tipos de un SGBD.

6. Lista y describe los SGBD más populares e indica si son comerciales o gratuitos con licencia libre.

Sistema Gestores de Bases de Datos

7. Hazte una cuenta Oracle y descarga e instala Oracle Database 18c Express Edition (establece una contraseña para las cuentas de SYS, SYSTEM y PGADMIN) y Oracle SQL Developer 20 y ejecuta éste último conectándolo con la base de datos Oracle. Ayúdate de este videotutorial.

8. Descarga y ejecuta el instalador de PostgreSQL 12 con los parámetros por defecto y, durante el proceso, establece una constraseña segura para el superusuario postgres. No es necesario ejecutar Stack Builder (el gestor de plugins de PostgreSQL). Por último, ejecuta PgAdmin 4 y establece una contraseña para PgAdmin 4 (master password). Recuerda, has establecido dos contraseñas. Una para el superusuario postgres y otra para PgAdmin 4.

9. Descarga y ejecuta el instalador de MariaDB con los parámetros por defecto y, durante el proceso, establece una constraseña segura para el superusuario root. Ayúdate de estos videotutoriales: español, inglés.

Evaluación

Criterios de calificación

  • Descarga, instalación de Oracle Database 18c Express Edition y ejecución de Oracle SQL Developer: hasta 4 puntos.
  • Descarga, instalación de PostgreSQL 12 y ejecución de PgAdmin 4: hasta 3 puntos.
  • Descarga, instalación y ejecución de MariaDB 10: hasta 3 puntos.

Criterios de evaluación asociados

  • En las actividad 1 se evaluará todos los criterios de evaluación del resultado de aprendizaje 1 (RA1).
  • En las actividad 2 se evaluará todos los criterios de evaluación del resultado de aprendizaje 2 (RA3).
  • En las actividad 3 se evaluará todos los criterios de evaluación del resultado de aprendizaje 3 (RA3).
