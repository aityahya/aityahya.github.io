Sindicación de contenidos: RSS y Atom

Contenidos

  1. Sindicación web.
  2. Ámbitos de aplicación: noticias (El Mundo, El País, Las Provincias), meteorología (El Tiempo), eventos (The Art Wolf), etc.
  3. Fuentes o canales web: RSS, Atom.
  4. Estructura y sintaxis de RSS 2.0.1 y Atom 1.0.
  5. Validación de RSS y Atom con W3C Feed Validation Service.
  6. Funcionalidades de RSS 2.0.1 y Atom 1.0.
  7. Agregadores: Feedly, Liferea, etc.

3. ¿Cuáles son los elementos mínimos necesarios para crear y validar un canal web con formato RSS 2.0.1?

4. Crea y valida un canal web de ejemplo, tres ítems como mínimo, sobre hormigas con la tecnología RSS 2.0.1.

5. Crea y valida un canal web de ejemplo, tres ítems como mínimo, sobre ciencia con la tecnología RSS 2.0.1.

6. ¿Cuáles son los elementos mínimos necesarios para crear y validar un canal web con formato Atom 1.0?

7. Crea y valida un canal web de ejemplo, tres entradas como mínimo, sobre GNU/Linux con la tecnología Atom 1.0.

8. Crea y valida un canal web de ejemplo, tres entradas como mínimo, sobre ajedrez con la tecnología Atom 1.0.

9. Busca cinco agregadores populares, instala tu favorito y añade en él un canal web sobre noticias.
