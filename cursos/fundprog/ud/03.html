3. Condicionales

Actividades de iniciación

01. Crea un programa que determine si dos variables de tipo numérico son iguales o no.

02. Crea un programa que determine si un número entero es par o impar (pista: usar la operación %).

03. Crea un programa que determine si una variable de tipo numérico es positiva, negativa o cero.

04. Crear un programa que determine si un número es menor que 0, o que está entre 0 y 50, o entre 51 y 100 o por encima de 100.

05. Crea un programa que determine si una letra es vocal o consonante.

06. Crea un programa que determine si, dado sus tres ángulos, se puede formar un triángulo (pista: un triángulo se forma cuando sus tres ángulos suman 180º).

07. Crea un programa que determine a partir de dos números cuál es el mayor y cuál es el menor o que imprima que son iguales si así lo son.

08. Crea un programa que encuentre el número mayor a partir de tres números dados.

09. Crea un programa que imprima por orden alfabético tres cadena de caracteres o strings dados y que imprima si son o no iguales dichos tres strings.

10. Crea un programa que determine si una persona está en edad de trabajar.

Actividades

Realiza los siguientes scripts. Los datos de entrada se pasan como parámetros del script.

11. Convierte de centímetros a metros y vicerversa.

Ejemplos
Entrada Salida
5cm 0.05m
-2.235m -223.5cm
56.74cm 0.5674m

12. Determina si un entero (mayor que 0 y menor que 10 000) al leerlo contiene la palabra cinco.

Ejemplos
Entrada Salida
5 Sí.
15 No.
25 Sí.
515 No.
5515 Sí.
7557 No.

13. Determina en una partida finalizada de tres en raya si gana O, gana X o hay empate.

Ejemplos
Entrada Salida
X X O O X O O O X Gana X.
X O O X X O O X O Gana O.
O X O O X X X O X Empate.

Entrada: Los tres primeros números pertenecen a la primera fila de la matriz, los tres siguientes a la segunda y los tres últimos números a la última fila.

14. Dados dos enteros positivos, imprime el valor más cercano a 21 sin sobrepasarse. Imprime 0 si los dos enteros son mayores que 21.

Ejemplos
Entrada Salida
19 21 21
22 17 17
3 3 3
16 23 16
18 17 18
