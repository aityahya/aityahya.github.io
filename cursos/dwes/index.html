Recursos

  • Referencias:
    • XHTML 5.2.
    • CSS 3.
    • Javascript.
    • PHP.

  • Especificaciones: HTML, CSS, JavaScript (ECMAScript), PHP.

  • Manuales:
    • XHTML 5.2 y CSS 3: MCLibre.
    • JavaScript: Desarrollo Web
    • PHP: PHP.net, MCLibre.

  • Validadores:
    • XHTML 5.2: Total Validator (descargar aquí).
    • CSS 3: The W3C CSS Validation Service (servidor Jigsaw).
    • WCAG 2.0 AAA: Total Validator (descargar aquí).
    • JavaScript: Closure Compiler (optimizador).

  • Control de versiones:
    • Herramientas: Git (diagrama) (comandos básicos) (referencia).
    • Servicios: GitHub, Heroku (tutorial).

  • Otros:
    • Repositorio: milq.
    • Plantillas: HTML/CSS.
    • Actividades: ejercicios de PHP con soluciones.

  • PHP y seguridad básica:
    • Entrada de usuario: formularios (ejemplo final).
    • Filtros: tipos, ejemplos.
    • Expresiones regulares: preg_match, RegExr, Regex101.
    • Manejo y registro de errores: constantes, funciones (error_reporting, set_error_handler).
    • Excepciones: ampliar las excepciones.

  • PHP y bases de datos:
    • Conceptos: gestión de bases de datos, tutorial de SQL.
    • PHP por procedimientos para PostgreSQL (aprended con los ejemplos que vienen):
      • Conexión y cierre: pg_connect y pg_close.
      • Ejecución de consultas: pg_query o pg_prepare-pg_execute (recomendado).
      • Asignación de resultados de consultas a variables: pg_fetch_result, pg_fetch_assoc y pg_fetch_all.
    • PHP orientado a objetos para MariaDB (aprended con los ejemplos que vienen):
      • Conexión y cierre: mysqli::connect y mysqli::close.
      • Ejecución de consultas:
        • Básico: mysqli::query.
        • Recomendado: mysqli::prepare, mysqli_stmt::bind_param, mysqli_stmt::bind_result, mysqli_stmt::execute.
      • Asignación de resultados de consultas a variables: mysqli_result::fetch_row, mysqli_result::fetch_assoc y mysqli_result::fetch_all.
    • PHP con interfaz PDO para cualquier bases de datos (aprended con los ejemplos que vienen):
      • Conexión: PDO::__construct (conexiones y su administración en PDO).
      • Ejecución de consultas:
        • Básico: PDO::query.
        • Recomendado: PDO::prepare y PDOStatement::execute.
      • Asignación de resultados de consultas a variables: PDOStatement::fetch y PDOStatement::fetchAll.
