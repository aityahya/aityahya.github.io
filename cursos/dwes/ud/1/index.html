Actividades

1. Crea un documento en XHTML 5 y CSS 3 (o Bootstrap) que explique las diferencias entre un desarrollador front-end, back-end y full-stack. Después, valídalo en XHTML 5 y CSS 3 (si usas Bootstrap).

1. Crea un tutorial en XHTML 5 y CSS 3 que enseñe a instalar XAMPP paso a paso con texto y capturas tanto en Windows 10 como en GNU/Linux. Asimismo, el tutorial debe enseñar a ejecutar código PHP puro y código PHP embebido en una página XHTML 5 y CSS 3.

2. Crea los siguientes formularios con XHTML 5 y CSS 3 y que los datos recogidos se procesen en una página XHTML 5 con PHP 7 embebido:
a) Recogida y validación de usuarios: crea este (o este) formulario en XHTML 5 y CSS 3 y que los datos de dicho formulario se recojan, se validen y se muestren en usuario.php.
b) Generador de CV: el usuario introducirá en un formulario sus datos personales, su experiencia profesional, su formación, dominio de idiomas y otros datos que sean de interés. Los datos de dicho formulario se recogerán, se validarán y se mostrarán en cv.php.

3. Crea las siguientes aplicaciones web con XHTML 5, CSS 3 (o Bootstrap) y PHP 7:
a) Piedra, papel, tijeras, lagarto o Spock: el usuario elegirá una de esas cinco opciones, recibirá una respuesta al azar y comprobará si ha ganado, perdido o empatado contra la máquina. Las reglas pueden verse aquí.
b) Área y longitud de una circunferencia: el usuario introducirá el radio de una circunferencia y al pulsar en Enviar se calculará su área y longitud.
c) Informe de salud personal: el usuario introducirá su sexo, edad, altura y peso y al pulsar en Calcular se validarán los datos y se generará un informe de salud con el IMC y el metabolismo basal según la fórmula de la OMS.

4. Realiza los siguientes ejercicios en XHTML 5 con CSS 3 o Bootstrap:
a) ¿Qué es un control de versiones? ¿Qué diferencias hay entre un control de versiones centralizado y otro distribuido?
a) Si no tienes una cuenta, crea una cuenta en con un nombre de usuario profesional (por ejemplo, iniciales de tus apellidos, etc.).
b) Área y longitud de una circunferencia: el usuario introducirá el radio de una circunferencia y al pulsar en Enviar se calculará su área y longitud.
c) Informe de salud personal: el usuario introducirá su sexo, edad, altura y peso y al pulsar en Calcular se validarán los datos y se generará un informe de salud con el IMC y el metabolismo basal según la fórmula de la OMS.

Evaluación

Criterios de calificación

  • Calidad del código PHP (funcional, eficiente y claro): hasta 4 puntos.
  • Calidad del código HTML y CSS (estructurado, organizado y claro): hasta 2 puntos.
  • Principios de diseño web: diseño de interfaz, diseño gráfico web y experiencia de usuario (navegabilidad, interactividad, usabilidad, etc.): hasta 2 puntos.
  • Aplicación web validado en XHTML 5, WCAG 2.1 AAA y CSS 3: hasta 2 puntos.

Criterios de evaluación asociados

  • En las actividad 1 se evaluará todos los criterios de evaluación del resultado de aprendizaje 1 (RA1).
  • En las actividad 2 se evaluará todos los criterios de evaluación del resultado de aprendizaje 2 (RA3).
  • En las actividad 3 se evaluará todos los criterios de evaluación del resultado de aprendizaje 3 (RA3).
