1. Generación de documentos web con PHP

Contenidos

  • Especificaciones: HTML, CSS.
  • Referencias: HTML, CSS.
  • Validadores: Total Validator (XHTML 5.2), The W3C CSS Validation Service (CSS3).
  • Manuales en español: HTML/CSS, PHP.

Actividades

1. Crea un tutorial en XHTML 5.2 y CSS 3 que enseñe a instalar XAMPP paso a paso con texto y capturas tanto en Windows 10 como en GNU/Linux. Asimismo, el tutorial debe enseñar a ejecutar código PHP.

2. Crea las siguientes aplicaciones web con XHTML 5.2, CSS 3 y PHP 7.3:

a) Recogida y validación de usuarios: crea el siguiente formulario en XHTML 5.2 y CSS 3 y que los datos de dicho formulario se recojan, se validen y se muestren en usuario.php.

b) Generador de CV: el usuario introducirá sus datos personales, su experiencia profesional, su formación, dominio de idiomas y otros datos que sean de interés. La aplicación web generará un curriculum vitae (CV) tanto en HTML (más CSS) como en PDF.

3. Crea las siguientes aplicaciones web con XHTML 5.2, CSS 3 y PHP 7.3:

a) Informe de salud personal: el usuario introducirá su edad, altura, peso, medidas, actividad y todos los datos necesarios. La aplicación web generará un informe de salud tanto en HTML (más CSS) como en PDF. Un ejemplo puede encontrarse aquí.

b) Texto: el usuario...

c) Filtrador de MAC: el usuario introducirá la MAC de su ordenador, se guardará en un fichero de texto y confirmará la operación. Si la MAC ya existe en el fichero, no se insertará dicha MAC y la aplicación advertirá de dicha duplicidad.

Evaluación

  • Funciona completamente el inicio de sesión tanto como usuario como administrador (se tendrá que proporcionar al profesor credenciales de ejemplo): hasta 1 punto.
  • Una vez iniciada la sesión, funciona perfectamente el servicio proporcionado para cada rol: hasta 2 puntos.
  • Genera un informe completo y detallado en HTML (más CSS) recogiendo la información de una base de datos (PosgreSQL): hasta 1.5 puntos.
  • Genera un informe completo y detallado en PDF recogiendo la información de una base de datos (PosgreSQL): hasta 1.5 puntos.
  • Funciona completamente el cerrar la sesión como usuario: hasta 0.5 puntos.
  • Funciona completamente el registro o alta (teléfono, email, dirección, etc.) como usuario inactivo: hasta 0.5 puntos.
  • Una página escogida al azar está completamente validada en XHTML 5.2, WCAG 2.0 AAA y CSS3: hasta 1 punto.
  • Calidad del código PHP (funcional, eficiente y claro): hasta 1 punto.
  • Calidad del código PosgreSQL (funcional, eficiente y claro): hasta 1 punto.
  • En las actividad 1 se evaluará todos los criterios de evaluación del resultado de aprendizaje 1 (RA1).
  • En las actividad 2 se evaluará todos los criterios de evaluación del resultado de aprendizaje 2 (RA3).
  • En las actividad 3 se evaluará todos los criterios de evaluación del resultado de aprendizaje 3 (RA3).
