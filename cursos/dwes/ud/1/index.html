1. Generación de documentos web con PHP

Contenidos

  • Especificaciones: HTML, CSS.
  • Referencias: HTML, CSS.
  • Validadores: Total Validator (XHTML 5.2), The W3C CSS Validation Service (CSS3).
  • Manuales en español: HTML/CSS, PHP.

Actividades

1. Crea un tutorial en XHTML 5.2 y CSS que enseñe a instalar XAMPP paso a paso con texto y capturas tanto en Windows 10 como en GNU/Linux.

2. Crea una web dinámica (XHTML + CSS + PHP) que genere tanto un documento HTML final como un PDF final.

Elige uno de los siguientes temas:

a) Generador de CV: El usuario introducirá sus datos personales, su experiencia profesional, su formación, dominio de idiomas y otros datos que sean de interés. La aplicación web generará un curriculum vitae (CV) tanto en HTML (más CSS) como en PDF.

b) Informe de salud personal: El usuario introducirá su edad, altura, peso, medidas, actividad y todos los datos necesarios. La aplicación web generará un informe de salud tanto en HTML (más CSS) como en PDF. Un ejemplo puede encontrarse aquí.
