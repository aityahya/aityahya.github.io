7. Aplicaciones web híbridas con Laravel y Bootstrap

Realiza las siguientes actividades en Laravel y Bootstrap.

Actividades de iniciación (2 puntos)

1. Crea un sitio web que salude con ¡Hola, mundo!

2. Crea un sitio web personal con tres pestañas: Inicio, CV y Sobre mí.

3. Crea el siguiente formulario y que dichos datos se recojan, se validen de forma segura en servidor y se muestren en otra vista.

c) Informe de salud personal: el usuario introducirá su edad, altura, peso, medidas, actividad y todos los datos necesarios. La aplicación web generará un informe de salud. Un ejemplo puede encontrarse aquí. a) Piedra, papel, tijeras, lagarto o Spock: el usuario elegirá una de esas cinco opciones, recibirá una respuesta al azar y comprobará si ha ganado, perdido o empatado. Las reglas pueden verse aquí.

X. Crea una aplicación web CRUD sencilla sobre libros (id, ISBN, título, autor, editorial, edición, año). Ayúdate de este tutorial.

Actividades de desarrollo (2 puntos)

5. Crea una tienda CRUD de videojuegos (ISAN, título, desarrolladora, distribuidora, género, año) con las pestañas Inicio (presentación de la tienda y de la página web), Juegos (visualización de todos los juegos almacenados en la tienda), Sobre nosotros (descripción de la tienda), Contacto (datos de contacto y localización) y Admin. Ésta última con tres subpestañas: Añadir (inserción de un videojuego nuevo en la tienda), Editar (modificación de los campos de un videojuego determinado) y Borrar (eliminación de un videojuego).

6. Crea un balance sencillo de ingresos y gastos. Para obtener el balance, habrá que registrarse como usuario y después iniciar sesión. Una vez iniciada la sesión correctamente, el usuario introducirá, mediante formularios, una serie de ingresos y gastos, pulsará en Generar y, a continuación, verá un informe similar a éste.
