8. Crea las rutas

Antes de añadir las implementaciones de los métodos CRUD, hay que crear las rutas para el modelo/entidad Producto como recurso. En la documentación oficial, exactamente aquí, tienes toda la información más detallada.

Para empezar, abre el archivo routes/web.php y actualízalo:

<?php

use Illuminate\Support\Facades\Route;

use App\Http\Controllers\ProductoController;

Route::get('/', function () {
    return view('welcome');
});

Route::resource('productos', ProductoController::class);

En negrita está el código añadido (son dos líneas a añadir). Con el método estático resource() de la clase Route creas múltiples rutas para las diferentes acciones/métodos del recurso. Estas rutas están mapeadas a varios métodos de ProductoController que debes implementar más adelante:

  • GET /productos: mapeado al método index() de ProductoController.
  • GET /productos/create: mapeado al método create() de ProductoController.
  • POST /productos: mapeado al método store(). de ProductoController
  • GET /productos/{id}: mapeado al método show() de ProductoController.
  • GET /productos/{id}/edit: mapeado al método edit() de ProductoController.
  • PUT/PATCH /productos/{id}: mapeado al método update() de ProductoController.
  • DELETE /productos/{id}: mapeado al método destroy() de ProductoController.

También puedes implementarlo manualmente si así lo deseas:

Route::get('/productos', 'ProductoController@index')->name('productos.index');
Route::get('/productos/create','ProductoController@create')->name('productos.create');
Route::post('/productos','ProductoController@store')->name('productos.store');
Route::get('/productos/{id}','ProductoController@show')->name('productos.show');
Route::get('/productos/{id}/edit','ProductoController@edit')->name('productos.edit');
Route::post('/productos/{id}/update','ProductoController@update')->name('productos.update');
Route::get('/productos/{id}/delete','ProductoController@destroy')->name('productos.destroy');
