9. Implementa los métodos CRUD

9.4 Implementa D (Delete)

Vamos a implementar el método de borrado para eliminar un producto guardado en la base de datos.

Para ello, abre app/Http/Controllers/ProductoController.php y actualízalo:

public function destroy($id)
{
    $producto = Producto::find($id);
    $producto->delete();
}

En negrita está el código añadido. Vamos a comprobar que funciona. Iniciamos el servidor de desarrollo de Laravel: php artisan serve. Abrimos el navegador y entramos en http://localhost:8000/productos.
