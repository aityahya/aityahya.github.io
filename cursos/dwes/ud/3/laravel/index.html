Tutorial de Laravel

Este es un tutorial en español para crear la primera aplicación web con Laravel. Este tutorial se ha testado en Debian 10 y en Windows 10. Si usas Debian/Ubuntu, necesitarás ejecutar comandos con permisos de administrador, es decir, entrando con su o ejecutando con sudo (sudo es más recomendable).

CRUD básico de productos

  1. Instala XAMPP y Composer.
  2. Crea un proyecto Laravel en blanco.
  3. Configura la conexión a la base de datos.
  4. Crea un modelo y su fichero de migración.
  5. Actualiza el fichero de migración y migra.
  6. Define el modelo.
  7. Crea el controlador.
  8. Crea las rutas.
  9. Implementa los métodos CRUD:
    • Implementa C (Create).
    • Implementa R (Read).
    • Implementa U (Update).
    • Implementa D (Delete).
  10. Últimos retoques.
