Tutorial de Laravel

Este es un tutorial para Laravel en español para desarrollar aplicaciones web. Este tutorial se ha testado en Debian 10 y en Windows 10.

Si usas Debian/Ubuntu, quizás necesites ejecutar los comandos con permisos de administrador, es decir, entrando con su o ejecutando con sudo (ejecutando con sudo es más recomendable).

1. Instala XAMPP y Composer

  1. Instala XAMPP en la ruta que viene por defecto (C:\xampp en Windows u /opt/lampp/ en Debian/Ubuntu).
    • En Debian/Ubuntu, justo después, ejecuta sudo rm -rf /opt/lampp/htdocs/* y luego sudo chown -R usuario:usuario /opt/lampp/htdocs donde usuario es tu nombre de usuario. Así podrás trabajar en htdocs con tu usuario y no siendo administrador.
  2. Instala Composer en Windows (mediante el instalador) o en Debian/Ubuntu (con sudo apt-get install composer).
    • En Debian/Ubuntu, justo después, ejecuta sudo apt-get install php php-xml php-mysql para instalar PHP, el módulo de XML para PHP necesario y el módulo de MySQL necesario.

Verifica que todo funciona correctamente, para ello:

  1. Inicia el servidor XAMPP.
  2. Comprueba PHP en la terminal ejecutando php --version en Windows u /opt/lampp/bin/php --version en Debian/Ubuntu.
  3. Comprueba Composer en la terminal ejecutando composer --version en Windows o en Debian/Ubuntu.

2. Instala Laravel

Laravel es un framework web que sigue el patrón de arquitectura de software Modelo-vista-controlador (MVC).

Existen varias maneras de instalar Laravel, mi preferida es creando un proyecto de Laravel directamente desde Composer (Via Composer Create-Project); por tanto, en este punto no hay que realizar nada.
