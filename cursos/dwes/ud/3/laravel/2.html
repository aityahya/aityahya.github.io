Tutorial de Laravel

2. Crea un proyecto Laravel en blanco

Laravel es un framework web que sigue el patrón de arquitectura de software Modelo-vista-controlador (MVC).

Existen varias maneras de instalar Laravel, mi preferida es creando un proyecto de Laravel directamente desde Composer (Via Composer Create-Project).

Crea tu primer proyecto con composer create-project laravel/laravel nombre_de_mi_proyecto. Ejecuta composer create-project laravel/laravel crud para crear un proyecto en blanco llamado crud.

Comprueba que funciona Artisan

Artisan es la interfaz de línea de comandos de Laravel que proporciona comandos útiles para facilitarte el desarrollo de tu aplicación web.

A continuación, entra desde tu terminal entra en C:\xampp\htdocs\crud en Windows u /opt/lampp/htdocs/crud en Debian/Ubuntu y comprueba que te funciona Artisan, la interfaz de línea de comandos de Laravel, ejecutando php artisan.

Con Artisan también puedes comprobar la versión de Laravel instalada ejecutando php artisan --version.
