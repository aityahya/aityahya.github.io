7. Crea el controlador

Después de crear el modelo Producto, migrarlo y actualizarlo, crea el controlador.

Laravel facilita la creación de controladores y adición de métodos estilo CRUD.

A continuación, ejecuta:

php artisan make:controller ProductoController --resource --model=Producto

Este comando creará el controlador ProductoController y de forma predeterminada siete métodos como el indexar, almacenar, crear, actualizar, destruir, mostrar, editar.

Puedes ver el resultado en app/Http/Controllers/ProductoController.php. Automáticamente, como puedes observar, se ha creado la clase ProductoController que deriva de la clase Controller y que contiene métodos para realizar operaciones CRUD con el modelo Producto.

Ahora necesitamos proporcionar implementaciones para estos métodos.
