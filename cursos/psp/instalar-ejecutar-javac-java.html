Compilar y ejecutar en Java desde línea de comandos

Ejecutar códigos en Java desde línea de comandos permite: consumir menos recursos, funcionar más rápido y aprender cómo funciona el compilador y el intérprete de Java. Es ideal si no necesitamos crear un proyecto de mayor envergadura. Asimismo, hay que sumar todas las ventajas de usar línea de comandos.

Los pasos para compilar y ejecutar en Java son los siguientes:

  1. Instala JDK: si no tienes JDK instalado, usa el instalador para Windows 10 o ejecuta sudo apt-get install default-jdk para distribuciones GNU/Linux basadas en Debian.

  2. Descarga un ejemplo: descarga el código de ejemplo HelloWorld.java.

  3. Compila a Java bytecode: desde terminal, tanto en Windows como en GNU/Linux, entra en el directorio donde esté HelloWorld.java, compila con javac HelloWorld.java y comprueba que se ha generado HelloWorld.class.

  4. Ejecuta código bytecode: en el mismo directorio, ejecuta java HelloWorld y verás en pantalla Hello World!.

Si no reconoce los comandos en Windows 10, establece los siguientes alias:

Set-Alias -Name javac -Value 'C:\Program Files\Java\jdk1.8.0_221\bin\javac.exe'

Set-Alias -Name java -Value 'C:\Program Files\Java\jdk1.8.0_221\bin\java.exe'

Cambia el directorio de JDK por el correspondiente en tu PC.
