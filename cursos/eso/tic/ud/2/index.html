2. Sistemas operativos

Contenidos

  1. Arquitectura.
  2. Funciones y componentes.
  3. Normas de utilización (licencias).
  4. Configuración, administración y monitorización.

Actividades

1. Sigue el tour guiado de Ubuntu 14.04.

2. ¿Qué es el software privativo o propietario? Enumera cinco programas de código cerrado así como los enlaces a su página oficial.

3. ¿Qué es el software libre? Indica las ventajas de usar software libre en vez de software privativo o propietario.

4. ¿Qué es una distribución GNU/Linux? ¿Qué diferencia hay con respecto a Mac OS X y Microsoft Windows 10?

5 (grupo). Buscad las cinco distribuciones GNU/Linux qué más os agraden y escribid el porqué.
