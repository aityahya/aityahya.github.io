4. Redes

Contenidos

  1. Red de computadoras.
  2. Tipos de redes.
  3. Dispositivos de red o de interconexión.
  4. Dispositivos móviles.
  5. Tarjeta o adaptadores de red.

Actividades

1. Describe con tus propias palabras lo que entiendes por red de computadoras.

2. Enumera las diferentes formas de clasificar una red.

3. Explica las diferentes formas de conexión en la comunicación entre dispositivos digitales tanto por alcance (PAN, LAN, WAN...) como por tipo de conexión (medios guiados y no guiados). Acompaña una imagen para cada una de las formas y tipos.

4. Enumera los pasos a realizar para conectarte a una red inalámbrica wifi en: un ordenador con Windows 10, un ordenador con Debian GNU/Linux y un móvil con Android o iOS.

5. Encuentra diez aplicaciones de comunicación entre dispositivos como WhatsApp, Gmail, Telegram, Skype, etc. Acompaña una imagen y una descripción para cada aplicación.

6 (ampliación). Define lo que es una topología de red e indica sus tipos. Acompaña una imagen, una descripción y las ventajas e inconvenientes para cada tipo.

7 (ampliación). ¿Cómo montarías una LAN en un aula de informática? ¿Y una WAN?
