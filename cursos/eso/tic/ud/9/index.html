HERRAMIENTAS DE SELECCIÓN

06.png Examina las herramientas de selección de GIMP. A continuación, descarga tres imágenes de internet: un camaleón, un león y una mofeta y crea una imagen nueva de un inédito animal de tamaño 500 × 1000 píxeles cuyo cuerpo sea de un camaleón, la cabeza de un león y la cola de una mofeta. Para esta actividad utiliza las selecciones rectangular, elíptica y libre y las operaciones de cortar, copiar y pegar.

07.png Descarga tres imágenes de internet: un ratón, un perro salchicha y un zorro y crea una imagen nueva de un inédito animal de tamaño 500 × 1000 píxeles cuyo cuerpo sea de un perro salchicha, la cabeza de un ratón y la cola de un zorro. Utiliza las selección de tijeras inteligentes (pulsa la tecla Intro para finalizar las tijeras) y las operaciones de cortar, copiar y pegar.

08.png Usando la selección por color, elimina los colores oliva, óxido (rust), turquesa, ocre y lima de la siguiente imagen. Modifica el umbral para ampliar o reducir el rango de selección.

09.png Haz transparente el fondo de la siguiente imagen. Para ello, añade una capa de transparencia y, usando la selección difusa (o varita mágica), elimina el fondo o segundo plano. Modifica el umbral para ampliar o reducir el rango de selección y ayúdate del zoom. Para finalizar, escala la imagen reduciéndola al 25 %.

10.png Mira el siguiente vídeo que enseña a usar la selección del frente (o primer plano). Usando dicha selección, extrae la flor e insértala en la siguiente imagen.

HERRAMIENTAS DE TRANSFORMACIÓN

11.png Examina las herramientas de transformación. A continuación, transforma esta imagen en esta nueva. Para ello, selecciona, por ejemplo con el lazo, la boca y luego aplica la herramienta de perspectiva. Realiza lo mismo para cada uno de los ojos.

12.png Haz el siguiente puzle a partir de esta imagen. Para ello, selecciona, por ejemplo con la varita o por color, una de las piezas del puzle, cópiala, pégala, muévela a su posición y aplica una de las siguientes transformaciones: rotar, escalar, inclinar o voltear. Haz lo mismo para el resto de las piezas. Para finalizar, recorta la imagen por la línea de puntos.
