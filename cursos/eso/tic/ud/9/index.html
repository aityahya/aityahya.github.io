HERRAMIENTAS DE SELECCIÓN

07.png Examina las herramientas de selección de GIMP. A continuación, descarga tres imágenes de internet: un camaleón, un león y una mofeta y crea una imagen nueva de un inédito animal de tamaño 500 × 1000 píxeles cuyo cuerpo sea de un camaleón, la cabeza de un león y la cola de una mofeta. Para esta actividad utiliza las selecciones rectangular, elíptica y libre y las operaciones de cortar, copiar y pegar.

08.png Descarga tres imágenes de internet: un ratón, un perro salchicha y un zorro y crea una imagen nueva de un inédito animal de tamaño 500 × 1000 píxeles cuyo cuerpo sea de un perro salchicha, la cabeza de un ratón y la cola de un zorro. Utiliza las selección de tijeras inteligentes (pulsa la tecla Intro para finalizar las tijeras) y las operaciones de cortar, copiar y pegar.

09b.png Haz transparente el fondo de la siguiente imagen. Para ello, añade una capa de transparencia y, usando la selección difusa (o varita mágica), elimina el fondo o segundo plando. Modifica el umbral para ampliar o reducir el rango de selección.
