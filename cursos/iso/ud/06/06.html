6. Supervisión del rendimiento del sistema

Contenidos

  1. Herramientas de monitorización en Windows 10:
    • Task Manager (taskmgr).
    • Event Viewer (eventvwr).
    • Performance Monitor (perfmon).
    • Reliability Monitor (perfmon /rel).
    • Resource Monitor (resmon).
  2. Herramientas de monitorización en GNU/Linux:
    • Comandos básicos: uptime, time, top, ps, vmstat, free, who, w, etc.
    • Conjunto de comandos del paquete sysstat.
  3. Analizadores de sistemas operativos.

Actividades

Realiza las siguientes actividades en Windows 10 y/o en GNU/Linux y documéntalo en PDF o en HTML y CSS:

1. Identifica los diferentes tipos de procesos, trabajos (jobs) y/o eventos del sistema.

2. Utiliza herramientas de monitorización en tiempo real.

3. Monitoriza el rendimiento del sistema mediante registros de contador, o counters, usando Performance Monitor en Windows 10.

4. Planifica y configura registros de rendimiento usando Data Collector Sets de Performance Monitor en Windows 10.

5. Interpreta los registros de rendimiento creados en la actividad anterior.

6. Analiza el sistema para optimizar el rendimiento.
