Libro recomendado

Se aconseja la lectura del libro Python para todos.

Dicho libro trata la versión 2 de Python. Las diferencias con respecto a la versión 3 no son muchas. Por ejemplo, en Python 3 se usa la función print() en vez de la sentencia print.

Los capítulos propuestos para leer son: Tipos básicos, Colecciones, Control de flujo y Funciones.
