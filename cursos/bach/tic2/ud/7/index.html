Actividades

Recuerda

· Crea tu primer programa en Python siguiendo este tutorial.

Actividades de Python pidiendo datos al usuario con input()

01.py Realiza un conversor de Farenheit a Celsius.

02.py Calcula el área y el perímetro de un rectángulo dado sus lados.

03.py Concatena dos números, convierte el resultado a entero y súmale 5.

04.py Multiplica tres enteros, concatena el resultado con un 4 al final y divide el resultado por 2.

05.py Dado un entero, indica con un 0 si es par y con un 1 si es impar (pista: usa el operador %)

Actividades de Python para aprender condicionales if

06.py Crea un programa que determine si dos variables de tipo numérico son iguales o no.

07.py Crea un programa que determine si un número entero es par o impar (pista: usar la operación %).

08.py Crea un programa que determine si una variable de tipo numérico es positiva, negativa o cero.

09.py Crear un programa que determine si un número es menor que 0, o que está entre 0 y 50, o entre 51 y 100 o por encima de 100.

10.py Crea un programa que determine si una letra es vocal o consonante.

11.py Crea un programa que determine si, dado sus tres ángulos, se puede formar un triángulo (pista: un triángulo se forma cuando sus tres ángulos suman 180º).

12.py Crea un programa que determine a partir de dos números cuál es el mayor y cuál es el menor o que imprima que son iguales si así lo son.

13.py Crea un programa que encuentre el número mayor a partir de tres números dados.

14.py Crea un programa que dado tres cadena de caracteres o strings imprima si son o no iguales dichos tres strings.

15.py Crea un programa que determine si una persona está en edad de trabajar.

Actividades de Python para aprender bucles for (opcionales pero recomendados)

16.py Crea un programa que:

  • Sume todos los elementos (de tipo numérico) de un array, o lista, mediante un bucle.
  • Multiplique todos los elementos (de tipo numérico) de un array, o lista, mediante un bucle.

17.py Genera una tabla de multiplicar, del 1 al 10, de un número entero dado.

18.py Imprime en pantalla, solo usando un bucle, todos los números del 0 al 99 excepto el 3, el 6 y el 11.

19.py Multiplica todos los números impares existentes desde 1 hasta un número dado.

20.py Suma todos los dígitos de un número dado.
