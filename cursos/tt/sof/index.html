Actividades de sistemas operativos

1. Sigue el tour guiado de Ubuntu 14.04.

2. ¿Qué es el software privativo o propietario? Enumera cinco programas de código cerrado así como los enlaces a su página oficial.

3. ¿Qué es el software libre? Indica las ventajas de usar software libre en vez de software privativo o propietario.

4. ¿Qué es un sistema operativo? Enumera al menos diez sistemas operativos diferentes e incluye su logotipo.

5. ¿Qué es una distribución GNU/Linux? ¿Qué diferencia hay con respecto a Mac OS X y Microsoft Windows 10?

6 (grupo). Buscad las cinco distribuciones GNU/Linux qué más os agraden y escribid el porqué.

7. ¿Qué son los permisos de un archivo? Descúbrelo y explícalo con tus propias palabras. A continuación, investiga los diferentes permisos existentes de un archivo en Windows 10

8. Encuentra un videotutorial para instalar Windows 10.

Actividades de sistemas de numeración

9. ¿Qué es un sistema de numeración? ¿Qué es un sistema de numeración decimal? ¿Qué es un sistema de numeración binario? ¿Qué es un sistema octal? ¿Qué es un sistema hexadecimal? Explica y dibuja cómo se convierte:

• Un número en sistema decimal a sistema binario y viceversa.
• Un número en sistema binario a octal y viceversa.
• Un número en sistema binario a hexadecimal y viceversa.

10. Realiza en papel y bolígrafo los 30 ejercicios de estos apuntes.
