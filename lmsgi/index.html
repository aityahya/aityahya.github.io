LMSGI

Unidades didácticas

  1. Lenguaje de marcas extensible: XML.
  2. Lenguaje de marcas de hipertexto: HTML.
  3. Lenguaje de marcas de hipertexto extensible: XHTML.
  4. Hojas de estilo en cascada: CSS.
  5. Sindicación de contenidos: RSS y Atom.
  6. Lenguaje de definición de esquemas XML: XSD.
  7. Validación de documentos XML con respecto a esquemas XSD.
  8. Lenguaje extensible de hoja de estilo: XSLT.
  9. Transformación de documentos XML con XSLT.
  10. Sistemas de almacenamiento de información de documentos XML.
  11. Herramientas de tratamiento de documentos XML.
  12. Sistemas de gestión empresarial: Odoo.

Bibliografía de aula

  • Tutoriales: W3Schools, MCLibre, Tutorials Point.
  • Validadores: Total Validator (XHTML 5.2) y The W3C CSS Validation Service (CSS3).
  • Manuales (MCLibre): HTML y CSS, XSD, XSLT.
  • Plantillas: página en blanco en XHTML 5.2.
